Help

Navigation

Opens the Menu Screen.

Tap or swipe left/right to switch between heating circuits if there are several ones.

Heating program

Start or stop central heating or DHW heating.

Central heating, DHW heating and DHW circulation in accordance with the set time programs.

DHW heating and DHW circulation in accordance with the set time programs.

Heating and domestic hot water on reduced temperature for the current heating circuit. This mode overrides time programs.

Heating and domestic hot water on normal temperature for the current heating circuit. This mode overrides time programs.

No central heating and no DHW heating. Boiler and DHW system protected against frost.

Temporary energy saving economy mode.

Temporary heating with individually adjustable room temperature and DHW heating enabled.

The User can turn on economy mode if the operation mode is set to heating and there is no reduced time program currently available. There is no possibility to turn on the economy and party mode simultaneously. Detailed information concerning the behaviour of economy and party mode on heating devices supported by Vitotrol App can be found on the following Webpage: www.vitotrol-app.info.

Room temperature

Setting the required room temperature for the standard heating mode.

Data refresh

Trigger data refresh manually using the button.

When the application is refreshing data from heating system, the arrow will be animated. The arrow will return to its default state when the application successfully receives the current data.

Last Update

This timestamp shows when the last successful data refresh took place.

Status of connection to Vitodata

The colour of the Refresh button presents the status of the connection to Vitodata and the result of the most recent data refresh procedure.

Vitotrol mobile application is not connected to Vitodata.

Vitotrol mobile application is connected to Vitodata and the last data refresh finished successfully. The application was synchronized with the heating device.

Vitotrol mobile application is connected to Vitodata but the last data refresh failed. The application was not synchronized with the heating device.
